basic.print can also be used for some "hacks".
The following examples shows how to manipulate or read other elements; but any other JavaScript/jQuery exression can be used.
The final "" in first example prevents basic.print from printing anything at all.
The second example does not use the item data at all, but it is triggered by receiving an item value.
