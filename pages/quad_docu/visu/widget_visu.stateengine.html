To trigger a popup on longpress you have to add a popup.extpopup widget as shown below.
The popup could provide sliders to change the suspend time or other items related to the autoblind plugin.
See "stateengine" section in this docu for more information!
{% filter trim|escape|nl2br %}{% raw %} {{ visu.stateengine('stateengineicon2', 'item.automatik.stateengine_name', 'item.automatik.lock', 'item.automatik.release', config_stateengine_icons, config_stateengine_values, 'stateengine_popup') }} {{ popup.extpopup('stateengine_popup', ['switch', ['item1.lock', 'icon', [0,1], ['secur_open.svg','secur_locked.svg']]], ['flip', ['item1.flip', 'on', 'off', '0', '1']]) }} {% endraw %}{% endfilter %}
