Icons

{{ lib.smartvisu }} uses two kinds of icons

Dynamic Icons

Dynamic icons are based on svg-images. They are directly rendered in the browser and are the fastest way to show dynamic changes. Use the slider to see how the dynamic icon changes.

{{ basic.slider('slider1', 'bath_light_value', 0, 255, 1) }}

Static Icons

Static icons are base on png-images. Some color-variants are shipped in the icons directory. To make them responsable to the design use as path "icon0" for "normal" icons and "icon1" for the highlighted version.
icon0~'light_light.png'
icon1~'light_light.png'
